Flow2API Captcha Worker

Choose End user (managed API key + label) or Worker (dedicated registration key only). Only one mode is active at a time after you save.

Use wss:// for HTTPS APIs (e.g. flow-api.prismacreative.online). Plain ws:// on a public host is auto-upgraded to wss://; localhost stays ws://. Remote hosts require host access in the extension manifest; if you see WebSocket transport error after changing only the URL, reload the extension on chrome://extensions (Refresh) so updated permissions apply.
